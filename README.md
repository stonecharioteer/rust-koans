# Rust Koans Presentation

A lightweight, no-build presentation website for a 10-koan Rust talk.

## Run it

Open `index.html` in a browser.

## Files

- `index.html` — presentation layout
- `styles.css` — visual theme / slide styling
- `app.js` — koan data + presentation controls
- `koans-bg.png` — background artwork

## Add your koans

Edit the `koans` array in `app.js`.
Each item has:

- `title`
- `question`
- `code`
- `reflection`

## Controls

- `←` / `→`
- `Page Up` / `Page Down`
- `Space`
- `Home` / `End`
- On-screen buttons (Prev, Next, Fullscreen)
